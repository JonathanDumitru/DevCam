# Security Policy

## Supported Versions

Only the latest minor release is supported with security updates.

| Version | Supported |
| --- | --- |
| 1.2.x | Yes |
| < 1.2.0 | No |

## Reporting a Vulnerability

Email: jonathan@hinesdumitru.online

Please include:
- Affected version
- Reproduction steps
- Impact assessment
- Proof of concept (if available)
- Any relevant logs

## Response Timeline

- Initial response within 5 business days
- Status update within 10 business days
- Fix timeline depends on severity and complexity

## Severity Guidelines

- Critical: remote code execution, arbitrary file access, or data exposure
- High: privilege escalation or sandbox bypass
- Medium: denial of service or data integrity issues
- Low: minor issues with limited impact

## Disclosure Policy

We request coordinated disclosure to protect users.
