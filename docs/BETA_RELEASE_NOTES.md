# DevCam Beta Release Notes

## v1.2 (Beta) - 2026-01-26
Highlights:
- Stable continuous recording with a 15-minute rolling buffer.
- Menubar Save Clip slider for 1-15 minute exports (1-minute steps).
- Advanced clip export with custom start/end range selection.
- Recording quality selection (Low/Medium/High).
- Menubar visibility and zero-byte segment fixes applied.

Focus areas for testing:
- Export reliability at multiple durations.
- Performance across recording quality levels.
- Permission flow and restart-required settings behavior.
- Energy impact during Preferences and file actions.

Known issues:
- Console log spam: "Sample buffer has no image buffer".
- Energy spikes during Preferences or file actions.
